# Offline AI Web App

An elegant, local-first AI chat assistant web app that runs entirely offline using Ollama and your own models.

## Features
- Local chat interface with persona and model selection
- Supports vision models (image input)
- No data leaves your machine
- Modern, responsive UI

## Quick Start
1. **Clone or download this repository.**
2. Make sure you have Python installed (for the simple web server).
3. (Optional) Install and run Ollama with your preferred models.
4. In this folder, run:

	```powershell
	.\serve.ps1
	```
	This will start a local server at [http://localhost:8080](http://localhost:8080)

5. Open your browser and go to [http://localhost:8080](http://localhost:8080)

## File Overview
- `index.html` — Main web app UI
- `app.js` — App logic and chat handling
- `styles.css` — Modern, dark theme styles
- `serve.ps1` — PowerShell script to launch a local server
- `.gitignore` — Standard ignores
